# checker-bank
## Install & Usage?
```
git clone https://github.com/risqikhoirul/checker-bank.git
cd checker-bank
npm install
node index.js
```
After that select the bank list number.
By clicking on a number or letter.
Example [0, 1, 2, ... a, b, c...]
After that type the enter account number
